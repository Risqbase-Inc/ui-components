# RisqBase Design System v4.2

This folder contains the v4.2 design-system release. v4.2 is a document-level release; the corresponding implementation work in this package is tracked against `audit.md`.

| File | Purpose |
|------|---------|
| `spec.md` | The consolidated v4.2 specification (supersedes v4.1.1) |
| `plan.md` | The execution plan + source canon + original audit tracker (the contract for v4.2) |
| `audit.md` | Honest assessment of what shipped in v4.2 vs what is still owed |
| `implementation-plan.md` | Sprint plan for the engineering programme (S1–S6, ~12 weeks) |
| `v4.2.1-backlog.md` | Spec-patch items deferred from v4.2 |
| `notes/` | Design-decision rationale (e.g. `neutral-palette.md`) |

**Status as of the audit:** 28 of 50 audit rows complete; 22 are spec'd-but-not-yet-implemented (engineering programme, ~8–10 weeks). See `audit.md` for the row-by-row read.

For ownership and review routing, see `.github/CODEOWNERS` and the git log on each path.
