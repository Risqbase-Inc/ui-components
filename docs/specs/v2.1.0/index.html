v2.1.0 Pipeline · Claude Design · 2026-05-20

@risqbase-inc/ui-components v2.1.0 — closing the four mandatory gaps + three optional clean-ups.

The 2026-05-19 Claude Code verification identified four components missing from ui-components that block wholesale marketing-repo consumption: ImpactGraph, HeroVideo, CompliancePostureStrip + ClientGrid. Plus a verification ask on the 9 CitationChip variants. This pack is the design spec for all four, plus three optional items the orchestrator flagged as cheap and high-value: ArcDecoration primitive, Header dropdown polish, VideoEmbed placeholder token swap.

CEO ratified Path X — full quality bar, natural pace. Specs land 20 May; engineering builds 21 May → 03 June; v2.1.0 publishes 05 June. The 25 May marketing launch is independent — it runs against the V2 demo HTMLs.

2.3 Tokens

Every component resolves through the v4.3 chain. Zero net-new tokens proposed. The closest call is ImpactGraph's category palette — could in principle introduce --color-impact-graph-cat-* aliases, but the existing --color-chart-cat-{1..8} chain covers it cleanly. Build against chart-cat-* directly.

If implementation surfaces a new canonical pattern worth codifying in the DS spec, flag it in the implementing PR and we patch a separate v4.4 token release.

§6Out of scope for v2.1.0 · captured for future

  • DS v4.4 token patch — deferred. Nothing in this cycle warrants one. If implementation surfaces a new canonical pattern, flag in the PR.
  • ImpactGraph real-product data binding — primitive exposes the API; consumer wiring lives in RALIA app, not in ui-components. RALIA team picks up post-1-July.
  • HeroVideo analytics binding — component exposes onPlay/onProgress/onComplete callbacks; consumer-side concern.
  • PracticeCockpitSurface monolith — explicitly rejected. CompliancePostureStrip and ClientGrid stay independent. The Cockpit hero composes from both.
  • Header sub-menus + mobile nav — v2.2 candidates. 25 May launch site has no mobile-nav requirement per Sophie.
  • HeroVideo HLS / adaptive bitrate — v2.2 candidate; depends on Mux vs Cloudflare Stream hosting decision.
