Phase 2.4 · print

Engineered as if no colour ink existed. Even when it does.

Seven-tier ladder

100% — #000000 — headings, axis
80% — #333333 — body text
55% — #737373 — high band
60% — #999999 — subtle text
30% — #B3B3B3 — default rule
15% — #D9D9D9 — low band
5% — #F2F2F2 — very-low band

Two-carrier rule

A 2nd-generation photocopy can't tell 55% black from 80% black. The pattern fill keeps differentiating when chroma collapses.

5-band severity reads three ways: greyscale luminance + pattern density + the explicit band-label word.

Same belt-and-braces guarantee HC mode provides — the print system simply uses paper instead of pixels.
