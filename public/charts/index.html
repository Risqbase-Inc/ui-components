Palettes

Every chart consumes one of three palette families, depending on the dataset's nature:

  • Categorical (no ordering) — color.chart.cat.{1..8}. 8 distinct hues, AA-verified against white.
  • Sequential (one-dimensional, ordered) — color.palette.{risk,financial,operational}.{50..900}. Single-hue ramps. Choice cued by domain.
  • Diverging (two-dimensional from neutral) — color.palette.diverging-{warm,cool}.{n4..p4}. 9-stop with neutral midpoint.

See live palette swatches for hex values + AA ratios.

Three themes per chart type

Every chart type renders in light, dark, and print with values pulled from the token tree. HC (forced-colors) collapses hue to system keywords; differentiation moves to shape, dash, and pattern fill — see tokens/component/print-patterns.json for the four pattern-fill assets covering very-low / low / medium / high / very-high band severity.
